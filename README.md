## Installation

Instalação das lib's do projeto com npm

### Cloning of project

```bash
  git clone https://github.com/adsonsandias/Finance-Control.git
```

### Access the project folder and instalation of lib's

```bash
  cd Finance-Control
  
  npm install
```

### Running the project

```bash
  npm start
```
